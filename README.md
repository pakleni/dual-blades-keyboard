## Features
* Split, wireless
* ZMK firmware
* nice!nano mcu
* nice!view display
* rotary encoders
* Column-staggered 44 key layout (inspired by the [Void Ergo S](https://github.com/victorlucachi/void_ergo))
* Switch-agnostic - hybrid MX/Alps PCB footprints + combo plate cutout
* Floating PCB plate
* LiPo batteries
